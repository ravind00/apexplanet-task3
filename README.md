# Web Development Internship - Task 3 🚀

Welcome to my third web development project, created during my internship at **ApexPlanet Software Pvt. Ltd.** ## 📝 About the Project

This project takes a huge step forward by integrating advanced CSS animations, an interactive JavaScript-based quiz, and asynchronous data fetching from a live public API. All of this is wrapped in a highly polished Glassmorphism user interface.

## ✨ Key Features

* **Animated Gradient Background:** A smooth, continuously moving gradient background using CSS `@keyframes`.
* **Advanced Glassmorphism UI:** Frosted glass effects, hover transformations, and modern UI elements.
* **Interactive Tech Quiz:** A dynamically rendered multiple-choice quiz that validates answers, gives visual feedback (Red/Green), and tracks progression.
* **Live API Integration:** Uses JavaScript `fetch()` and `async/await` to retrieve and display random programming jokes from a live public API.

## 🛠️ Technologies Used

* **HTML5:** Semantic structure.
* **CSS3:** Advanced keyframe animations, CSS Grid, custom Google Fonts (Poppins), and backdrop-filter for the glass effect.
* **Vanilla JavaScript:** * DOM Manipulation & Event Handling
  * Data structures (Arrays & Objects for the quiz)
  * Asynchronous JavaScript (`async/await`, Try/Catch blocks)
  * Fetch API

## 🚀 How to Run

1. Clone or download this repository to your local machine.
2. Double-click the `task3.html` file to open it in any modern web browser.
3. **Note:** An active internet connection is required to fetch the live programming jokes from the API.
